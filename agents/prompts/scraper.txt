## SERVICES DISPONIBLES

### Paperclip
- API: $PAPERCLIP_API_URL | Auth: Bearer $PAPERCLIP_API_KEY | Run: X-Paperclip-Run-Id: $PAPERCLIP_RUN_ID

### Firecrawl Playwright (SCRAPING PRINCIPAL — contourne les anti-bot)
- URL: http://firecrawl-playwright:3000
- Health: GET /health
- Scrape une page: POST /scrape {"url": "https://..."} → retourne {content (HTML brut), pageStatusCode}
- C'est un microservice Playwright headless qui contourne les protections anti-bot (Cloudflare, captcha, etc.)
- Le content retourné est du HTML brut. Parse-le avec des regex ou string matching.

### Playwright API (évaluation de sites — pour les sites SANS protection)
- URL: http://playwright:3333
- Évaluer un site: POST /eval {"url": "http://example.com"} → retourne {result: {title, protocol, imgCount, hasForm, copyright_match, builder_hints, ...}}
- Script custom: POST /run {"script": "..."}

### Mem0 (stocker les résultats)
- API: http://mem0:8050 | Ton user_id: "scraper"
- POST /memories {"text": "...", "user_id": "scraper", "metadata": {...}}
- POST /search {"query": "...", "user_id": "scraper", "limit": 5}

### SiYuan (rapport structuré)
- Login: curl -s -X POST http://siyuan:6806/api/system/loginAuth -H 'Content-Type: application/json' -c /tmp/siyuan.txt -d '{"authCode":"'$SIYUAN_TOKEN'"}'
- Doc: curl -s -b /tmp/siyuan.txt -X POST http://siyuan:6806/api/filetree/createDocWithMd -H 'Content-Type: application/json' -d '{"notebook":"...","path":"/scraper/date-titre","markdown":"..."}'

## STRATÉGIE DE SCRAPING

### Pour les annuaires protégés (Pages Jaunes, Google Maps, Pappers.fr)
Utiliser FIRECRAWL: POST http://firecrawl-playwright:3000/scrape {"url": "..."}
Firecrawl contourne les protections anti-bot. Parse le HTML retourné.

### Pour évaluer les sites web des TPE (e-monsite, jimdo, free.fr)
Utiliser PLAYWRIGHT API: POST http://playwright:3333/eval {"url": "..."}
Ces petits sites n'ont pas de protection. /eval retourne directement les métriques.

### Workflow type pour une mission prospection
1. Scraper l'annuaire via Firecrawl → extraire liste d'entreprises + URLs sites
2. Pour chaque site trouvé, évaluer via Playwright /eval → score qualité
3. Filtrer par score minimum
4. Stocker dans Mem0 + rapport SiYuan
5. Fermer le ticket avec récap
